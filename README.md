# 🔗 Button Tooltip - Botão de Compartilhamento Interativo

Um botão de "Share" (Compartilhar) elegante e moderno que, ao ser clicado, revela um menu de ícones de redes sociais com um efeito de tooltip animado. Ideal para adicionar interatividade e estilo minimalista a qualquer interface web.

## 🌟 Descrição

Este projeto utiliza técnicas avançadas de CSS para criar um botão de ação que alterna entre os estados "Share" e "Cancel". Ao ativar o botão, um popup surge suavemente acima dele, contendo links para Facebook, Twitter, Instagram e YouTube.

**Destaques visuais:**
- **Animação de Tooltip:** O menu de redes sociais aparece com um efeito de fade-in e desliza levemente para cima.
- **Toggle de Texto:** O botão muda dinamicamente de "SHARE" para "CANCEL" através de pseudo-elementos CSS.
- **Efeitos de Hover:** Cada ícone de rede social possui uma cor personalizada que é ativada no hover, além de um leve movimento para cima.
- **Estética Clean:** Design baseado em bordas arredondadas (pills) e sombras suaves sobre um fundo azul vibrante.

## 🛠️ Funcionalidades Técnicas

- **Lógica Pure CSS:** Controle de exibição via `checkbox` e seletor `~`.
- **Pseudo-elementos:** Uso de `::before` para criar a seta do tooltip e alternar o texto do botão.
- **Transições Suaves:** Movimentos controlados pela propriedade `transition` para uma experiência de usuário fluida.
- **Layout Flexível:** Alinhamento horizontal dos ícones usando Flexbox.
- **Interatividade sem JS:** Todas as interações de clique e exibição são feitas apenas com CSS.

## 🚀 Tecnologias Utilizadas

- **HTML5:** Estrutura básica e inputs.
- **CSS3:** Estilização, animações, transformações e lógica de estado.
- **Font Awesome:** Ícones vetoriais de alta qualidade.
- **Google Fonts:** Tipografia Poppins.

## 📂 Como Usar

1. Baixe os arquivos do projeto.
2. Certifique-se de que o diretório `css/` contém o arquivo `style.css`.
3. Abra o `index.html` em seu navegador.
